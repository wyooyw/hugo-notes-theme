# Accelerating sparse matrix-vector multiplication on GPUs using bit-representation-optimized schemes.
## 主要思想
稀疏在GPU上效果不好，是因为其计算强度低，瓶颈在显存带宽上。所以针对稀疏格式的索引进行压缩，来提升速度。
要求解压步骤可以非常快地在GPU上执行。
